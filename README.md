## ğŸš€ Setup & Local Development

### Prerequisites
* Node.js & npm installed
* Akun [Supabase](https://supabase.com/)

### Installation
1. **Clone Repository**
   ```bash
   git clone [https://github.com/cexyzp/PILKETOS_2026.git](https://github.com/cexyzp/PILKETOS_2026.git)
   cd PILKETOS_2026

2. **Install Dependencies**
   ```bash
   npm install

3. **Konfigurasi Environment**
   ```bash
   Buat file .env di root direktori:
   API_KEY=your_supabase_anon_key
   DB_URL=your_supabase_url

4. **Jalankan Aplikasi**
   ```bash
   node app.js

5. **Database Setup (Supabase SQL)**
    Buka SQL Editor di dashboard Supabase kamu dan jalankan query berikut: 
```bash
    CREATE TABLE public.candidates (
       id bigint PRIMARY KEY,
       name text NOT NULL,
       class text,
       photo text,
       vision text,
       missions text[],
       created_at timestamp with time        zone DEFAULT now() NOT NULL
   );
```
```bash 
   CREATE TABLE public.votes (
        id bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
        created_at timestamp with time zone DEFAULT now() NOT NULL,
        candidate_id bigint REFERENCES public.candidates(id) ON DELETE CASCADE
    );
```
